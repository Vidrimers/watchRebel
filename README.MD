watchRebel — это социальная платформа для тех, кто любит фильмы и сериалы. Здесь можно вести собственные коллекции, оценивать просмотренное, писать отзывы и находить людей со схожими вкусами.

## Что можно делать

- 🎬 **Коллекции** — вести списки фильмов и сериалов, отмечать просмотренные эпизоды
- ⭐ **Оценки** — ставить оценки от 1 до 10
- 📝 **Отзывы** — писать текстовые рецензии на фильмы и сериалы
- 📋 **Списки** — создавать собственные категории («Любимые», «Посмотреть позже» и другие)
- 👥 **Друзья** — добавлять друзей, смотреть их активность и списки
- 📰 **Лента** — следить за активностью друзей в ленте
- 💬 **Сообщения** — общаться с друзьями в личных сообщениях
- 📌 **Стена** — писать посты на своей стене и на стенах друзей, прикреплять изображения
- 🎯 **Поиск** — искать фильмы, сериалы и пользователей
- 🔔 **Уведомления** — получать оповещения о действиях друзей в Telegram и на сайте
- 🎨 **Темы** — выбирать из множества тем оформления

## Как начать

1. Найдите бота **@watchRebel_bot** в Telegram
2. Нажмите «Начать» — бот пришлёт ссылку на сайт
3. Перейдите по ссылке — вход автоматический
4. Начните добавлять фильмы и сериалы в свои списки

## Возможности

### Для зрителя

- Просмотр подробной информации о фильмах и сериалах (данные TMDb)
- Отслеживание прогресса просмотра сериалов
- Экспорт списков в файл
- Реферальная система — приглашайте друзей

### Для общения

- Посты на стене с изображениями и галереями
- Комментарии к постам и фотографиям с вложенностью
- Реакции на посты
- Личные сообщения с файлами и вложениями
- Упоминание друзей через @

## Технологии

- **Frontend:** React, Vite, Redux Toolkit
- **Backend:** Node.js, Express, SQLite
- **Бот:** Telegram Bot API (polling)
- **Данные:** TMDb API
- **Деплой:** PM2, Nginx, SSL (Let's Encrypt)
