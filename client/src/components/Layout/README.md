# Layout компоненты

Набор компонентов для создания основного layout приложения watchRebel.

## Компоненты

### UserPageLayout

Основной layout для страницы пользователя. Содержит левый блок (Wall) и правый блок (Sidebar).

**Props:**
- `children` - контент для левого блока (Wall)
- `user` - объект пользователя

**Пример использования:**
```jsx
import { UserPageLayout } from './components/Layout';

<UserPageLayout user={currentUser}>
  <Wall posts={posts} />
</UserPageLayout>
```

### Sidebar

Правый блок управления с поиском, аватаром, навигацией и настройками.

**Props:**
- `user` - объект пользователя

**Содержит:**
- Поисковую строку
- Аватар и имя пользователя
- Навигационное меню
- Уведомления
- Ссылку на настройки

### Footer

Футер приложения, прижатый к низу страницы.

**Props:** нет

## Адаптивность

Все компоненты адаптивны и корректно отображаются на:
- Desktop (> 1024px) - двухколоночный layout
- Tablet (768px - 1024px) - одноколоночный layout
- Mobile (< 768px) - оптимизированный одноколоночный layout

## CSS переменные

Компоненты используют CSS переменные из `index.css`:
- `--bg-primary` - основной фон
- `--bg-secondary` - вторичный фон (карточки)
- `--text-primary` - основной текст
- `--text-secondary` - вторичный текст
- `--accent-color` - акцентный цвет
- `--border-color` - цвет границ

## Максимальные ширины

- Wall (левый блок): 1300px
- Sidebar (правый блок): 500px
- Общая ширина контента: 1820px (1300 + 500 + 20px gap)
