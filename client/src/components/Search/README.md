# Search Components

Компоненты для поиска фильмов, сериалов и пользователей.

## Компоненты

### SearchBar

Компонент поисковой строки с preview (быстрым предпросмотром результатов).

**Особенности:**
- Debounce 300ms для оптимизации запросов
- Preview с первыми 5 результатами
- Переход на полную страницу поиска по Enter
- Автоматическое закрытие preview при клике вне компонента
- Поддержка навигации к результатам

**Использование:**
```jsx
import SearchBar from './components/Search/SearchBar';

<SearchBar />
```

### SearchPage

Полная страница поиска с фильтрами.

**Особенности:**
- Фильтры: Все, Пользователи, Фильмы, Сериалы
- Отображение количества результатов по каждому типу
- Адаптивная сетка результатов
- Карточки пользователей и медиа
- Обработка состояний загрузки и ошибок

**Использование:**
```jsx
import SearchPage from './pages/SearchPage';

// В роутере
<Route path="/search" element={<SearchPage />} />
```

## Интеграция с Redux

Компоненты используют `mediaSlice` для управления состоянием поиска:

- `searchMedia` - async thunk для выполнения поиска
- `setSearchQuery` - action для установки текущего запроса
- `clearSearch` - action для очистки результатов

## Требования

- React Router для навигации
- Redux Toolkit для состояния
- TMDb API для данных о фильмах/сериалах
